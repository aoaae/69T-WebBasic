จุดประสงค์การเรียนรู้

  1. 1

    อธิบายการใช้งานแท็ก <a> เพื่อสร้างลิงก์ไปยังหน้าเว็บหรือแหล่งต่าง ๆ

  2. 2

    แทรกรูปภาพด้วยแท็ก <img> พร้อมกำหนดคุณสมบัติพื้นฐาน

  3. 3

    ตกแต่งลิงก์และรูปภาพด้วย CSS เพื่อความสวยงามและน่าใช้งาน

🏷

แท็กที่ใช้บ่อย

<a> href target <img> src alt width height
📋

กิจกรรมฝึกปฏิบัติ

  • 1 สร้างลิงก์อย่างน้อย 3 แบบ
  • 2 แทรกรูปภาพอย่างน้อย 2 รูป
  • 3 ใส่ alt ให้กับรูปภาพทุกภาพ
  • 4 ใส่ CSS เปลี่ยนสีลิงก์เมื่อ hover
  • 5 ตกแต่งรูปภาพด้วย border-radius หรือ box-shadow
  • 6 บันทึกไฟล์ชื่อ “link-image.html”

เกร็ดความรู้

แท็ก <a> ใช้สำหรับสร้างลิงก์ (Hyperlink) เพื่อเชื่อมโยงไปยังหน้าเว็บ ไฟล์ หรือที่อยู่อีเมลต่าง ๆ โดยใช้แอตทริบิวต์ href ระบุปลายทางของลิงก์ แท็ก <img> ใช้สำหรับแทรกรูปภาพ โดยใช้แอตทริบิวต์ src ระบุแหล่งที่อยู่ของไฟล์รูปภาพ และ alt ระบุข้อความอธิบายรูปภาพ ซึ่งสำคัญต่อการเข้าถึงได้ (Accessibility) และช่วยแสดงข้อความเมื่อไม่สามารถโหลดรูปได้ • CSS สามารถใช้ตกแต่งลิงก์ด้วย link color, เอฟเฟกต์เมื่อ hover, ขนาดของรูปภาพ, ขอบ (border), มุมโค้ง (border-radius) และเงา (box-shadow) เพื่อให้หน้าเว็บสวยงามและน่าใช้งานมากยิ่งขึ้น
