Workshop: Heading

ฝึกใช้แท็ก <h1> – <h4> เพื่อจัดโครงสร้างหัวข้อของหน้าเว็บ

Course Overview

What You Will Learn

  • Understand heading levels (<h1> – <h6>)
  • Design a clear content hierarchy for a web page
  • Improve readability and structure using headings

Requirements

  • Basic knowledge of HTML
  • A simple text editor (VS Code, Notepad++, etc.)
  • Web browser (Chrome, Edge, Firefox)

✏️คำแนะนำ: ให้ลองเปลี่ยนข้อความหัวข้อเป็นภาษาไทย หรือแก้ชื่อหัวข้อให้เข้ากับวิชาที่นักศึกษากำลังเรียนอยู่ได้เลย

Module 1: HTML Basics

Understanding Tags

Block vs Inline Elements

  • Block elements: กินพื้นที่เต็มบรรทัด เช่น <div>, <p>, <h1>
  • Inline elements: อยู่ในบรรทัด เช่น <span>, <a>, <strong>

Writing Your First HTML Page

  • สร้างโครง HTML พื้นฐาน (<!DOCTYPE html>, <html>, <head>, <body>)
  • เพิ่ม <h1> เป็นหัวข้อหลักของหน้า
  • ตามด้วย <p> เพื่ออธิบายเนื้อหา

✅ แบบฝึกหัด: ให้ผู้เรียนสร้างไฟล์ my_first_page.html ที่มี <h1> 1 อัน และ <h2>–<h4> อย่างน้อยอย่างละ 1 อัน

Module 2: CSS Fundamentals

CSS Selectors

  • Element selector: เช่น h1 { color: red; }
  • Class selector: เช่น .card { background: white; }
  • ID selector: เช่น #main-title { text-align: center; }

Styling Text

  • เปลี่ยนสีตัวอักษรด้วย color
  • กำหนดฟอนต์ด้วย font-family
  • ปรับขนาดด้วย font-size

🎨 ลองต่อยอด: ให้ผู้เรียนเปลี่ยนสี h1, h2, h3 ให้คนละสี และสังเกตความแตกต่างเวลาอ่าน
